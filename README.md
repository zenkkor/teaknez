# teaknez.com

Static site for **Tea Knez Coaching** — osebni, karierni in poslovni coaching v Sloveniji.

## Stack

Pure static HTML/CSS/JS, no framework. Pages are generated by a small Python
script (`build.py`) from shared templates so headers, footers, and shared
fragments stay in sync.

## Local preview

```bash
python3 build.py             # regenerate all HTML
python3 -m http.server 8765  # serve at http://localhost:8765
```

## Structure

```
build.py                # template + page builder (run after content changes)
index.html              # generated
o-meni.html             # generated
storitve.html           # generated
storitve/*.html         # service detail pages (generated)
blog.html               # generated
blog/*.html             # blog posts (generated)
kontakt.html            # contact page with web3forms (generated)
politika-zasebnosti.html
piskotki.html
sitemap.xml             # generated
robots.txt              # generated
assets/css/main.css     # all styles
assets/js/main.js       # nav + contact form behavior
assets/images/          # site imagery
```

## Contact form

The contact form posts to [web3forms](https://web3forms.com/) (`api.web3forms.com/submit`)
with AJAX. The access key lives in `build.py` (`page_contact`). Inline success /
error messaging is handled by `assets/js/main.js`.

## Deploying to GitHub Pages

1. Push to `main`.
2. Repo → Settings → Pages → Source: **Deploy from a branch** → `main` / `/`
   (root).
3. Add custom domain (e.g. `www.teaknez.com`); GitHub will create a `CNAME`
   file.
